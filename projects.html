Projects

This is a more curated list of projects I have worked on.

Press VR

Press VR is a virtual reality training application developed using Unreal Engine. It was my senior project and was made under the supervision of a client from HP. The program trains users how to use the HP PageWide Industrial press, a large industrial printer, in a 3D virtual environment. This was a group project, but I got my hands on just about everything. I worked on Player controls, Object interaction, Level design, Lighting, Materials, and More!

C++ Ray Tracer

A general ray tracer written in C++ based on the book Ray Tracing in One Weekend, by Peter Shirley. Features include: a vec3 class, a simple camera, and various materials (diffuse, metal, clear). Generates and renders a random scene of spheres with different sizes and materials.

OpenGL Helicopter Game

This is a small helicopter game made using OpenGL as a final project for my Intro to Computer Graphics course. Features include: a player controlled helicopter, a textured environment, rings for the player to fly through, and a camera that changes orientation based on player position.
